### contoh pemakaian

```
node cariAwal.js aba --tunggal
```

menghasilkan hanya kata tunggal, seperti:  
aba  
aba-aba  
abad  
abadi  
...dst  


```
node cariAwal.js aba --majemuk
```

menghasilkan hanya kata majemuk, seperti:  
abad keemasan  
abad komputer  
abad modern  
abad pertengahan  
...dst

node cariAwal.js aba
